Come incorporarla in Google Sites

Metodo consigliato: pubblica questo file HTML su un hosting statico che generi un URL pubblico HTTPS, poi in Google Sites usa Inserisci → Incorpora → URL.

Alternative pratiche: GitHub Pages, Netlify, Cloudflare Pages oppure Google Drive/Apps Script se già li usi come supporto.

Perché non basta il file locale: Google Sites incorpora in modo stabile soprattutto contenuti pubblici raggiungibili via URL.

Uso interno: puoi anche aprire l’HTML direttamente nel browser come archivio personale offline.

Suggerimento operativo: se vuoi, posso anche trasformare questa app in una versione con logo, colori personalizzati, esportazione in CSV o area per aggiungere nuovi prompt dall’interfaccia.
